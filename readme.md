<h1 align="left">Yike.io</h1>

Source code of https://yike.io.

## Installation

First, download the dependencies using Npm:

```js
npm install
```

Then, run the project:

```js
npm run serve
```

## Local Config

First, copy the `.env.example` to `.env.local`:

```shell
cp .env.example .env.local
```
